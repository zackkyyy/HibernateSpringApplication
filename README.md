# HibernateSpringApplication

First you should download the database finaltaskjava.sql as it the database connected to the program and initialized with some data to run the project 

[Home Page](http://localhost:8080)
## Some Apis provided by the application 

[View all characters](http://localhost:8080/api/characters)     
[View all users](http://localhost:8080/api/users)    
[View all classes](http://localhost:8080/api/classes)    
[Find a user by adding id number to the end of path](http://localhost:8080/api/user/1)    
[Find a character by adding id number to the end of path](http://localhost:8080/api/character/1) 
[Find a class by name "add a name to the end of path"](http://localhost:8080/api/class/Warrior)          
[Add a user](http://localhost:8080/api/addUser)         
[Add a character](http://localhost:8080/api/addCharacter)
